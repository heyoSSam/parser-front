# parser-front

## Installation

To get started with this project, follow these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/heyoSSam/parser-front
cd parser-front
```

### 2. Install Dependencies

```bash
npm install -g pnpm
pnpm install
```

### 3. Start Server

```bash
pnpm run dev
```
